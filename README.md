# Spatial-Information-Project-Project-PFC
Objetivo:  Investigar os fundamentos científicos e tecnológicos necessários para substituir telas por sistemas de informação espacial tridimensional.  Pergunta central:  É possível transformar o espaço físico em um meio computacional e informacional capaz de armazenar, exibir e permitir manipulação de informação sem superfícies tradicionais?
