# Projeto Interface Espacial

Objetivo:

Investigar os fundamentos científicos e tecnológicos necessários para substituir telas por sistemas de informação espacial tridimensional.

Pergunta central:

É possível transformar o espaço físico em um meio computacional e informacional capaz de armazenar, exibir e permitir manipulação de informação sem superfícies tradicionais?

Áreas de estudo:

* Matemática
* Física
* Eletromagnetismo
* Ondas
* Teoria da Informação
* Computação
* Sistemas Complexos

Status atual:

* Python: em desenvolvimento
* Primeira simulação de ondas concluída
* Estudo de ondas iniciado
